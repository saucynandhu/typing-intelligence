# phase-9-visualizations

## Status: PLANNED / NEXT

## Goals
Visualize user performance and model confidence using interactive charts.

### Planned Features
- **WPM/Accuracy Over Time**: Line charts showing historical performance.
- **Finger-Heatmap (Mock)**: Visualization of key-level latency derived from the sequence model.
- **Strategy Distribution**: Pie chart showing how often the user triggers Recovery vs. Challenge modes.
- **Model Confidence Bar**: Visual indicator of how well the engine "knows" the user's typing profile.

## Notes
Will likely use `Recharts` or `Chart.js` for React integration.
